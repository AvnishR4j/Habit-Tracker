## 📖 About The Project

**Monthly Habit Tracker** is a lightweight, browser-based productivity tool that helps you plan and track your daily habits throughout the month. It gives you a clear, visual overview of your progress with a real-time clock, a completion percentage, and a monthly calendar grid — all in a clean, distraction-free interface.

At the end of every month, the tracker **automatically resets**, giving you a fresh start for the next cycle.

---

## ✨ Features

- ✅ **Add & Delete Habits** — Quickly create your daily tasks and remove the ones you no longer need
- 🔥 **Daily Streak Tracking** — Keep track of your consistency across the month using a calendar grid view
- ⏰ **Live Clock** — A real-time digital clock displayed on screen to keep you grounded in the moment
- 📊 **Completion Percentage** — See at a glance how many tasks you've completed out of your total (e.g. *3 of 5 tasks complete → 60%*)
- 🔄 **Monthly Auto-Reset** — Your habit slate wipes clean at the start of every new month, so you can begin fresh
- 📆 **Monthly Calendar View** — A full month grid (Sun–Sat) to visualize your habit journey

## 🚀 Getting Started

To run this project locally:

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- [Git](https://git-scm.com/) installed on your machine

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd YOUR_REPO_NAME
   ```

3. **Open `index.html` in your browser**
   ```bash
   # On macOS
   open index.html

   # On Linux
   xdg-open index.html

   # On Windows
   start index.html
   ```

> No build tools or installations needed — it runs purely in the browser!
